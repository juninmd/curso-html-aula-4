# Curso HTML Aula 4

**Description:** A standard software project.

**Installation:**

1.  Clone the repository: `git clone https://github.com/seuocursos/curso-html-aula-4.git`
2.  Navigate to the project directory: `cd curso-html-aula-4`
3.  Install dependencies: `npm install` (or equivalent for your package manager)

**Usage:**

*   **Understanding the Files:**
    *   `.gitignore`: Specifies files to ignore.
    *   `basico.html`:  Basic HTML structure.
    *   `bower.json`:  Bower configuration.
    *   `cadastro.html`:  Login page.
    *   `cadastro.js`:  Login form JavaScript.
    *   `index.html`:  Main HTML file.
    *   `jquery.mask.js`: jQuery mask implementation.
    *   `login.html`: Login page content.
    *   `nascimentos.html`:  Nascimentos page content.
    *   `puro.html`:  Puro example page.
    *   `renovate.json`:  Renovate.json file.
    *   `trabalho.txt`:  A simple text file.
*   **Basic Page Structure:**
    *   `index.html` serves as the main landing page.
    *   `cadastro.html` handles user registration.
    *   `cadastro.js` processes form submissions.
    *   `login.html` displays the login page.
    *   `nascimentos.html` displays a set of natal forecasts.
    *   `puro.html` contains a simple example page.
    *   `renovate.json` manages the project's state.
    *   `trabalho.txt` is a placeholder.
*   **Development:** Use the browser to view the pages.  For example: `index.html` in your browser.
